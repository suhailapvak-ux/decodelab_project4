# Quick Start Guide - General Knowledge Quiz

Get started with the General Knowledge Quiz in just a few seconds! ⚡

## 🏃 Quick Start

### Step 1: Ensure Python is Installed
```bash
python --version
# or
python3 --version
```

Requires Python 3.6 or higher.

### Step 2: Run the Quiz
```bash
python knowledgequiz.py
```

### Step 3: Answer the Questions
- Read each question carefully
- Type your answer and press Enter
- Get instant feedback on your answer
- Answers are case-insensitive (PARIS = paris = Paris)

### Step 4: View Your Results
After all 5 questions, you'll see:
- Your total score (e.g., 4/5)
- Your percentage (e.g., 80%)
- Your performance rating (Excellent, Very Good, Good, or Needs Improvement)

## 📊 Performance Scale

| Score | Rating                |
|-------|----------------------|
| 5/5   | ⭐ Excellent         |
| 4/5   | 👍 Very Good         |
| 3/5   | 😊 Good              |
| <3/5  | 📚 Needs Improvement |

## 💡 Tips

✅ **Spaces don't matter** - The program removes leading/trailing spaces
✅ **Case doesn't matter** - Uppercase, lowercase, or mixed case all work
✅ **Take your time** - No time limit, answer at your own pace
✅ **Learn as you go** - See the correct answer if you get it wrong

## 🆘 Troubleshooting

### Problem: "python: command not found"
**Solution**: 
- Use `python3` instead of `python`
- Or add Python to your system PATH
- Visit [python.org](https://www.python.org/downloads/) to install

### Problem: The quiz doesn't respond to input
**Solution**:
- Make sure you press Enter after typing your answer
- Check that your terminal has focus

### Problem: Wrong answer feedback doesn't appear
**Solution**:
- Run the script in a terminal (not an editor)
- Ensure your Python version is 3.6+
