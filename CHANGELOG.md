# Changelog

All notable changes to the General Knowledge Quiz project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-07

### Added

- ✨ Initial release of General Knowledge Quiz
- 📝 5 engaging general knowledge questions covering:
  - Geography (Capital of France)
  - Astronomy (Red Planet)
  - Wildlife (National animal of India)
  - History (Father of Computer)
  - Nature (Largest ocean)
- 📊 Score tracking and calculation
- 💯 Percentage calculation with decimal precision
- 🎯 Performance rating system (Excellent, Very Good, Good, Needs Improvement)
- ✅ Real-time feedback for each answer ("Correct!" / "Wrong!")
- 🔤 Case-insensitive answer validation
- 📌 Whitespace handling with `.strip()` method
- 💻 User-friendly console interface
- 📚 Comprehensive code documentation with comments
- 📖 Professional README.md with usage instructions
- 📦 requirements.txt file
- 📜 MIT License
- 🤝 CONTRIBUTING.md for contributors
- 📝 CHANGELOG for version tracking
- 🙈 .gitignore for clean repository

### Features

- Seven Python concepts demonstrated:
  - Variables (score tracking)
  - Input/Output operations
  - Conditional logic (if-else statements)
  - String methods (.strip(), .lower())
  - Arithmetic operations
  - F-string formatting
  - Code comments and documentation

### Code Quality

- Clean, readable code structure
- Comprehensive inline comments
- Following PEP 8 standards
- No external dependencies required
- Beginner-friendly code style
